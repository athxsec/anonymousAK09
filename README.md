## 👨‍💻 About Me

I'm `athxsec`, a driven cybersecurity student and ethical hacker based in Maharashtra, India. My journey into offensive security spans from mastering network penetration with **Kali Linux** to physical hardware exploitation and custom RF development. I thrive on dissecting systems, uncovering vulnerabilities, and building robust security solutions.

-   🎓 Actively pursuing my **12th Grade (HSC)** while constantly expanding my expertise in advanced cybersecurity domains.
-   🛡️ Proficient in using **Kali Linux** for **network penetration testing**, **web application security assessments**, and **vulnerability research**.
-   🐍 Crafting powerful security tools, automation scripts, and web interfaces using **Python** and **Streamlit**.
-   📡 Deeply involved in **hardware hacking**, **RF analysis**, and **firmware reverse engineering** with custom devices like **ESP32 (M5Stick), CC1101, and nRF24L01**. I love bringing concepts to life with **Arduino**.
-   🖥️ I maintain a **homelab** for simulating attack scenarios, testing defenses, and experimenting with server configurations.
-   🌐 Passionate about open-source contributions and collaborating on impactful security projects.
-   📫 Connect with me directly on [Instagram](https://instagram.com/athxsec)!
